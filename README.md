## Setup
### Requirements
- Ruby 2.7.3
- Rails 6.1.6.1
- Node 16.16.0
- Postgresql Databse


### Usage
1. Create the database
> rails db:create

2. Perform migrations
> rails db:migrate

3. Install gems and dependencies
> bundle install
> yarn install

4. Populate the database
> rails db:seed
